# 链表元素去重复

## 83 - Remove Duplicates from Sorted List

### 原题概述

Given a sorted linked list, delete all duplicates such that each element appear only _once_.

**Example 1:**

```text
Input: 1->1->2
Output: 1->2
```

**Example 2:**

```text
Input: 1->1->2->3->3
Output: 1->2->3
```

### 题意和分析

给一个链表，检查里面的元素，去重，让每个元素只出现一次。

```text
Input: 1->2->3->3->4->4->5

Input: 1->1->1->2->3

```

### 题意和分析
