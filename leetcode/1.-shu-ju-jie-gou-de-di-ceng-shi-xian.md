# 2. 基础数据结构底层实现

### 链表

```java
public class LinkedList<E> {//支持泛型

    /*
     * 将结点设置成一个私有的类，
     * 让外部不能访问，用户并不需要了解链表的底层实现，因此对外部用户屏蔽实现细节
     * */
    private class Node {
        //类的成员变量做成public，让外部LinkedList类可以随意访问和修改，而不需要getter和setter
        public E e;
        public Node next;

        public Node (E e, Node next) {
            this.e = e;
            this.next = next;
        }

        //用户只传入一个参数e
        public Node (E e) {
            this(e, null);//调用上面两个参数的方法，其中next为null
        }

        //用户什么都不传
        public Node() {
            this(null, null);//调用上面两个参数的方法，e和next均为null
        }

        @Override
        public String toString() {//每个结点的打印结果
            return e.toString();
        }
    }
}
```
